Paideia LMS にログインしたので、ユーザープロフィールを更新してアカウントをパーソナライズしましょう！

<PreCheck>
  - ユーザープロフィール設定にアクセスする
  - アバター画像をアップロードする
  - プロフィール情報（自己紹介、名前など）を更新する
</PreCheck>

## プロフィールにアクセスする

<Steps>
1. ユーザープロフィールに移動します。通常、上部ナビゲーションバーの名前やアバターをクリックすると見つかります。

2. 「プロフィールを編集」または「設定」をクリックして、プロフィール編集ページにアクセスします。

   以下のフィールドが表示されるはずです：
   - **名**と**姓**
   - **メールアドレス**（通常は読み取り専用）
   - **自己紹介** - 自分についての短い説明
   - **アバター** - プロフィール写真
   - **テーマ** - ライトまたはダークモードの設定
</Steps>

## アバターをアップロードする

<Steps>
3. アバター画像をアップロードします：

   - アバターアップロード領域または「アバターを変更」ボタンをクリックします
   - コンピューターから画像ファイルを選択します（一般的な形式：JPG、PNG、GIF）
   - 必要に応じて画像をトリミングまたは調整します
   - 変更を保存します

   :::tip[アバターのベストプラクティス]
   - 最良の結果を得るには、正方形の画像を使用します
   - 推奨サイズ：200x200ピクセル以上
   - ファイルサイズを適切に保ちます（2MB未満）
   :::

4. アバターがプロフィールとナビゲーションバーに正しく表示されることを確認します。
</Steps>

## プロフィール情報を更新する

<Steps>
5. プロフィールの詳細を更新します：

   - **名**と**姓**：これらが正しいことを確認します
   - **自己紹介**：自分自身、役割、または興味についての簡単な説明を追加します
   - **テーマ**：希望するテーマ（ライトまたはダークモード）を選択します

6. 「保存」または「プロフィールを更新」ボタンをクリックして変更を保存します。

7. プロフィールページを表示して変更を確認します。
</Steps>

:::note[プロフィールの表示]

プロフィール情報は、システム内の他のユーザーに表示されます。アバターと名前は通常すべてのユーザーに表示されますが、その他の詳細はプライバシー設定によって異なる場合があります。
:::

おめでとうございます！プロフィールがパーソナライズされ、準備が整いました！
