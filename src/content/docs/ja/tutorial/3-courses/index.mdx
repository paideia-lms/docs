アクティビティモジュールを作成し、プロフィールをパーソナライズしたので、コンテンツをコースに整理し、学生のアクセスを管理しましょう！

## これから学ぶこと

このユニットでは、Paideia LMS で**コース**を作成して管理する方法を学びます。以下を作成します：

- セクションとカテゴリで整理されたコース
- セクションにモジュールを追加してコースコンテンツを作成
- 登録を通じて学生のアクセスを管理
- グループを使用して学生を組織化

その過程で、コースコンテンツを構造化し、学生のアクセスを管理し、協調的な活動のために学生をグループに組織化する方法を学びます。
