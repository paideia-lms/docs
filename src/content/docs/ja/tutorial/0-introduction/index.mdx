<Lede> このチュートリアルでは、インストールから最初のコース作成まで、最初のLMSインスタンスをセットアップしながら、Paideia LMSの主要な機能を学んでいきます！🚀 </Lede>

その過程で、以下のことをおこないます。
- Paideia LMSのインストールと設定
- データベースとストレージのセットアップ
- 最初のコースの作成
- コースへのアクティビティモジュールの追加
- ユーザーと登録の管理
- 本番環境用のLMSの設定

インストール前にPaideia LMSを試したい場合は、[デモサイト](https://demo.paideialms.com)でインストールなしですべての機能を探索できます。

:::note
事前に設定されたインスタンスでPaideia LMSを探索したい場合は、[demo.paideialms.com](https://demo.paideialms.com)にアクセスしてPaideia LMSの動作を確認できます。
:::
