これから何を作成するのかわかったところで、必要なツールを準備していきましょう！

このユニットでは、Paideia LMSのインストール、データベースとストレージのセットアップ、環境の設定方法を説明します。すでにPaideia LMSをインストールして設定している場合は、[ユニット2](/ja/tutorial/2-users/)に進んでください。

:::tip[インストール前に試す]

インストール前にPaideia LMSを試したいですか？[demo.paideialms.com](https://demo.paideialms.com)にアクセスして、インストールなしですべての機能を探索できます。

デモは、Hetzner CPX21インスタンス（3 vCPU、4GB RAM、80GBローカルディスク）でホストされ、毎日のバックアップ付きで、月額$10未満のコストで、Paideia LMSの効率的なリソース使用を実証しています。
:::

## これから向かう場所

このユニットでは、**Paideia LMSをダウンロードしてインストール**し、**データベースとストレージをセットアップ**し、**環境を設定**します。

次のことを学びます：
- Paideia LMSバイナリのインストール
- PostgreSQLデータベースのセットアップ
- S3互換ストレージの設定
- 環境変数の設定
- Paideia LMSインスタンスへのアクセス

インストールが完了したら、次のユニットで最初のコースを作成する準備が整います！
