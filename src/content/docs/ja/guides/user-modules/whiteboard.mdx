**ホワイトボード**モジュールは、講師が Excalidraw JSON として保存された描画と図表コンテンツを作成できる読み取り専用のアクティビティモジュールです。このモジュールタイプは、視覚的な図表、フローチャート、マインドマップ、その他のグラフィックコンテンツを提示するのに最適です。

## 概要

ホワイトボードモジュールは、ユーザーの対話や提出を必要とせずに視覚的なコンテンツを提示する情報提供型（読み取り専用）モジュールです。以下に最適です：

- 視覚的な図表とフローチャート
- マインドマップと概念マップ
- 技術的な図面と図表
- インタラクティブな視覚的説明

## ホワイトボードモジュールの作成

ホワイトボードモジュールを作成するには：

1. コースに移動します
2. 「モジュールを追加」または「アクティビティを作成」をクリックします
3. モジュールタイプから「ホワイトボード」を選択します
4. ホワイトボードのタイトルと説明を入力します
5. Excalidraw エディターを使用して視覚的なコンテンツを作成します
6. 準備が整ったら、下書きとして保存するか公開します

## Excalidraw エディター

ホワイトボードモジュールは Excalidraw を使用します。これは、以下をサポートする強力なオープンソースの描画ツールです：

- **描画ツール**：ペン、形状、矢印、線
- **テキスト要素**：ラベルと注釈を追加
- **形状ライブラリ**：矩形、円、菱形など
- **レイヤー**：要素をレイヤーに整理
- **エクスポートオプション**：PNG、SVG、JSON 形式
- **コラボレーション**：リアルタイムのコラボレーション編集（講師向け）

## モジュールステータス

ホワイトボードモジュールは標準のモジュールライフサイクルに従います：

- **下書き**：コンテンツが作成および編集中；作成者と許可されたユーザーのみに表示
- **公開済み**：コンテンツが学生の準備が整いました；登録学生に表示
- **アーカイブ済み**：コンテンツが保存されていますが、もはやアクティブではありません；学生には非表示

## ベストプラクティス

ホワイトボードモジュールを作成する際：

- **明確なラベルを使用**して、図表の異なる部分を識別します
- **適切な色を選択**して、視覚的な区別を向上させます
- **図表をシンプルに保つ**ことで、学生を圧倒しないようにします
- **複数のホワイトボードで一貫したスタイルを使用**します
- **複雑な図表に凡例を追加**します
- **バックアップ目的で画像としてエクスポート**します

## ホワイトボードの表示

学生は、インタラクティブなビューアーでホワイトボードコンテンツを表示でき、以下が可能です：

- 詳細な表示のためにズームイン/アウト
- 異なる領域を探索するためのパン
- ホワイトボードを画像としてダウンロード
- 必要に応じてホワイトボードを印刷
