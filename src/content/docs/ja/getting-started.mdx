:::caution[重要な注意事項]
このプロジェクトは積極的に開発中であり、まだ本番環境での使用には準備ができていません。

リソースが限られているため、バージョン1.0以前のドキュメントはAIによって自動生成されており、完全に正確ではない可能性があります。実際のシステム動作と照らし合わせて情報を確認し、不一致があれば報告してください。ドキュメントは継続的に改善しており、あなたのフィードバックが改善に役立ちます。
:::

:::note
Paideia LMSの構築と維持を支援する貢献者とスポンサーを積極的に探しています。開発者、教育者、またはオープンソースの教育技術をサポートすることに興味のある組織であれば、ぜひお聞かせください。
:::

<CardGrid>
  <Card title="Paideia LMSとは？" icon="laptop">
    Paideia LMSは、[Bun](https://bun.sh)と[React Router v7](https://reactrouter.com/)で構築された、モダンで軽量な学習管理システムです。単一の実行ファイルとして、管理とデプロイが非常に簡単に設計されています。詳細は[ホワイトペーパー](/ja/whitepaper/)をご覧ください。
  </Card>

  <Card title="クイックスタート" icon="star">
    数秒で起動できます！事前ビルドされたバイナリをダウンロードし、[インストールガイド](/ja/install-and-setup/)でLMSを開始しましょう。または、インストールなしでPaideia LMSを探索できる[デモサンドボックスを試す](https://demo.paideialms.com)こともできます。
  </Card>

  <SplitCard title="Paideiaをインストール" icon="rocket">
    <Tabs syncKey="paideia-install">
      <TabItem label="macOS ARM64">
        ```sh
        curl -L -o paideia https://github.com/paideia-lms/paideia/releases/latest/download/paideia-macos-arm64
        chmod +x paideia
        xattr -c paideia        
        ./paideia
        ```
      </TabItem>
      <TabItem label="Linux ARM64">
        ```sh
        curl -L -o paideia https://github.com/paideia-lms/paideia/releases/latest/download/paideia-linux-arm64
        chmod +x paideia        
        ./paideia
        ```
      </TabItem>
      <TabItem label="Linux x64">
        ```sh
        curl -L -o paideia https://github.com/paideia-lms/paideia/releases/latest/download/paideia-linux-x64
        chmod +x paideia
        ./paideia
        ```
      </TabItem>
    </Tabs>

    [インストールガイド](/ja/install-and-setup/)には、macOS、LinuxでのPaideiaのインストール、およびDocker Composeの使用に関するステップバイステップの手順が記載されています。
