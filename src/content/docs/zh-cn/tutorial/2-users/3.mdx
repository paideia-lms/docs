现在您已创建了页面模块，让我们创建一个**白板**模块！白板非常适合创建视觉图表、流程图和其他图形内容。

<PreCheck>
  - 创建白板模块
  - 使用 Excalidraw 编辑器创建视觉内容
  - 保存并发布您的白板
</PreCheck>

## 创建白板模块

<Steps>
1. 导航到您想要添加模块的模块库或课程。

2. 点击"添加模块"或"创建活动"按钮。

3. 从模块类型列表中选择"白板"。

   您应该看到以下字段：
   - **标题** - 您的白板模块的名称
   - **描述** - 白板内容的简要说明
   - **白板编辑器** - 用于创建视觉内容的 Excalidraw 编辑器
</Steps>

## 创建视觉内容

<Steps>
4. 为您的白板模块输入标题，例如："系统架构图"

5. 添加描述："此白板显示了 Paideia LMS 的架构。"

6. 使用 Excalidraw 编辑器创建您的视觉内容：

   Excalidraw 编辑器提供：
   - **绘图工具**：笔、形状、箭头、线条
   - **文本元素**：添加标签和注释
   - **形状库**：矩形、圆形、菱形等
   - **图层**：将元素组织到图层中
   - **颜色**：为不同元素选择颜色
   - **导出选项**：PNG、SVG 和 JSON 格式

7. 创建一个简单的图表：
   - 绘制一个标记为"前端"的矩形
   - 绘制另一个标记为"后端"的矩形
   - 用箭头连接它们
   - 添加文本标签来描述连接
   - 使用颜色来区分不同的组件
</Steps>

## 保存并发布您的白板

<Steps>
8. 在预览中查看您的白板内容。

9. 选择您的模块状态：
   - **草稿**：如果您想稍后继续编辑，则保存为草稿
   - **已发布**：如果您准备好让学生使用，则发布

10. 点击"保存"或"发布"以保存您的白板模块。

11. 验证您的白板是否出现在您的模块库或课程中。
</Steps>

:::tip[白板最佳实践]

- 使用清晰的标签来识别图表的不同部分
- 选择适当的颜色以获得更好的视觉区分
- 保持图表简单，以免让学生感到不知所措
- 在多个白板之间使用一致的样式
- 为复杂图表添加图例
:::

恭喜！您已创建了第一个白板模块！
