## 创建新课程

<Steps>
1. 导航到课程创建区域。通常可以通过在课程管理界面中点击"创建课程"或"新建课程"找到。

2. 输入课程信息：
   - **标题**：为您的课程输入描述性标题，例如："Paideia LMS 简介"
   - **描述**：添加课程涵盖内容的简要说明
   - **类别**（可选）：选择或创建类别以组织您的课程

3. 点击"创建"或"保存"以创建您的课程。

   :::tip[课程类别]
   类别有助于组织您的课程目录。您可以创建嵌套类别以实现分层组织。例如，"计算机科学" → "编程" → "Web开发"。
   :::
</Steps>

## 使用章节组织您的课程

<Steps>
4. 创建课程章节以组织您的内容：

   章节提供了一种逻辑地构建课程内容的方法。您可以创建：
   - **顶级章节**：主要单元或周
   - **嵌套章节**：主章节内的子章节

5. 创建您的第一个章节：
   - 在您的课程中点击"添加章节"或"新建章节"
   - 输入章节名称，例如："第1周：介绍"
   - 可选地添加描述
   - 保存章节

6. 创建嵌套章节（可选）：
   - 在章节内，您可以创建子章节
   - 例如，在"第1周：介绍"内创建：
     - "第1天：入门"
     - "第2天：基本概念"

   :::tip[章节组织]
   章节可以嵌套以创建分层结构。这允许您在多个级别组织内容，例如周 → 天 → 主题。
   :::

7. 根据需要重新排序章节：
   - 使用拖放或排序控件来排列章节
   - 章节和模块在每个章节内排序，控制学生遇到内容的顺序
</Steps>

## 课程结构

您的课程现在具有如下结构：

```
Paideia LMS 简介
├── 第1周：介绍
│   ├── 第1天：入门
│   └── 第2天：基本概念
├── 第2周：高级主题
└── 第3周：最终项目
```

每个章节可以包含：
- **子章节**（嵌套的子章节）
- **活动模块**（页面、测验、作业等）
