现在您有了带章节的课程，是时候将您的活动模块添加到课程中了！模块链接到课程，允许您在多个课程中重用相同的模块。

<PreCheck>
  - 将活动模块链接到课程章节
  - 配置课程特定的模块设置
  - 在章节内重新排序模块
</PreCheck>

## 将模块链接到课程章节

<Steps>
1. 导航到要添加模块的章节。

2. 在章节中点击"添加模块"或"链接模块"。

3. 从您的模块库中选择一个模块：
   - 您将看到您可以访问的模块列表
   - 选择要添加到此章节的模块
   - 模块将链接到课程章节

4. 配置模块顺序：
   - 指定模块在章节内的顺序
   - 您稍后可以使用拖放或排序控件重新排序模块

   :::tip[模块可重用性]
   单个活动模块可以链接到多个课程。这意味着您可以创建一次模块并在不同课程中重用它，节省时间并确保一致性。
   :::
</Steps>

## 配置课程特定设置

<Steps>
5. 为模块配置课程特定设置：

   当模块添加到课程时，它成为具有课程特定设置的**课程模块**。这些设置允许您自定义模块在此特定课程中的行为方式。

6. 设置模块特定选项：
   - **自定义模块名称**：为此课程覆盖默认模块名称
   - **截止日期**（用于交互式模块）：设置提交截止日期
   - **开放/关闭日期**：控制模块何时对学生可用
   - **截止日期**：允许延迟提交但带有惩罚

   :::note[课程模块设置]
   课程模块设置仅影响模块在此特定课程中的显示和行为方式。它们不会更改模块的默认配置或影响其在其他课程中的使用。
   :::

7. 保存您的课程模块设置。
</Steps>

## 添加多个模块

<Steps>
8. 向您的课程添加更多模块：

   - 重复该过程以向不同章节添加模块
   - 您可以将同一模块添加到多个章节或课程
   - 在每个章节内逻辑地组织模块

9. 根据需要重新排序模块：
   - 使用拖放或排序控件
   - 模块在每个章节内排序，控制学习顺序
</Steps>

## 模块组织示例

您的课程现在可能如下所示：

```
Paideia LMS 简介
├── 第1周：介绍
│   ├── 第1天：入门
│   │   ├── 欢迎页面（页面模块）
│   │   └── 课程概述测验（测验模块）
│   └── 第2天：基本概念
│       ├── 概念页面（页面模块）
│       └── 练习测验（测验模块）
└── 第2周：高级主题
    ├── 高级概念（页面模块）
    └── 最终作业（作业模块）
```
