## 获取所需的先决条件

### 操作系统

Paideia LMS 运行在 **macOS**（ARM64 或 x64）或 **Linux**（ARM64 或 x64）上。确保你可以访问兼容的操作系统。

### 终端

你将使用**命令行（终端）**来下载、安装和运行 Paideia LMS，并配置环境。

你可以通过操作系统的本地终端程序访问命令行。常见的应用程序包括 **Terminal**（macOS/Linux）、**命令提示符**（Windows）和 **Termux**（Android）。其中一些可能已经在你的设备上了。

### PostgreSQL 数据库

Paideia LMS 需要 **PostgreSQL** 数据库。你需要以下之一：
- 本地 PostgreSQL 安装
- 访问远程 PostgreSQL 数据库
- Docker Compose（包含 PostgreSQL）

要检查是否在本地安装了 PostgreSQL，请运行：

```sh
psql --version

# 输出示例
psql (PostgreSQL) 18.0
```

如果你没有安装 PostgreSQL，你可以：
- 在本地安装 PostgreSQL
- 使用 Docker Compose（我们将在安装步骤中介绍）
- 使用云 PostgreSQL 服务

### S3 兼容存储

Paideia LMS 需要用于文件存储的 **S3 兼容存储**。你需要以下之一：
- MinIO（S3 兼容存储）- 包含在 Docker Compose 中
- S3 兼容服务（AWS S3、DigitalOcean Spaces 等）
- 本地 MinIO 安装

### 代码编辑器（可选）
