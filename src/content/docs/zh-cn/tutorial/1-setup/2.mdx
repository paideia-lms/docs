## 下载 Paideia LMS 二进制文件

单二进制安装是启动 Paideia LMS 最快的方法。Paideia LMS 作为包含所需一切的单个可执行文件分发。

<Steps>
1. 下载适用于你平台的二进制文件：

   <Tabs>
     <TabItem label="macOS ARM64">
       ```bash
       curl -L -o paideia https://github.com/paideia-lms/paideia/releases/latest/download/paideia-macos-arm64
       chmod +x paideia
       xattr -c paideia
       ```
     </TabItem>
     <TabItem label="Linux ARM64">
       ```bash
       curl -L -o paideia https://github.com/paideia-lms/paideia/releases/latest/download/paideia-linux-arm64
       chmod +x paideia
       ```
     </TabItem>
     <TabItem label="Linux x64">
       ```bash
       curl -L -o paideia https://github.com/paideia-lms/paideia/releases/latest/download/paideia-linux-x64
       chmod +x paideia
       ```
     </TabItem>
   </Tabs>

2. 通过检查版本来验证安装：

   ```bash
   ./paideia version
   ```

   你应该看到 Paideia LMS 版本号，确认二进制文件正常工作。
</Steps>

:::tip[替代方案：Docker Compose]

如果你更喜欢使用 Docker Compose 进行包含 PostgreSQL 和 MinIO 的完整设置，请参阅[安装和设置指南](/zh-cn/install-and-setup/#docker-compose-安装)中的 Docker Compose 说明。
:::
