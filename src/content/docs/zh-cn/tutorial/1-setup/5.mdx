## 启动 Paideia LMS

现在你的环境已经配置好了，你可以启动 Paideia LMS 服务器。

<Steps>
1. 启动 Paideia LMS 服务器：

   ```bash
   ./paideia server
   ```

   或者简单地：

   ```bash
   ./paideia
   ```

   服务器将启动并在配置的端口上监听（默认：后端 API 为 `3001`，前端为 `3000`）。

2. 等待服务器启动。你应该在终端中看到 Paideia LMS 正在运行的确认信息。

   :::tip[服务器启动]
   如果有待处理的迁移，服务器将在启动时自动运行数据库迁移。
   :::
</Steps>

## 访问 Paideia LMS

<Steps>
3. 打开浏览器并导航到 `http://localhost:3000`（或通过 `FRONTEND_PORT` 环境变量配置的端口）。

   你应该看到 Paideia LMS 登录页面。

4. 首次访问时，系统会提示你创建管理员账户。按照屏幕上的说明创建管理员用户。

   :::caution[首次设置]
   确保记住你的管理员凭据！你需要它们来登录和管理你的 Paideia LMS 实例。
   :::

5. 创建管理员账户后，登录以访问 Paideia LMS 仪表板。
</Steps>

恭喜！你现在已经运行并可以访问 Paideia LMS 了！

## 验证安装

<Steps>
6. 探索 Paideia LMS 界面：

   - 检查仪表板
   - 浏览不同的部分
   - 验证所有功能都可以访问

7. 检查 API 端点：

   导航到 `http://localhost:3001` 以验证 API 正在运行（你应该看到 API 文档或健康检查响应）。
</Steps>

:::tip[运行 Paideia LMS]

当 Paideia LMS 服务器正在运行时，你可以：
- 在 `http://localhost:3000` 访问 Web 界面
- 在 `http://localhost:3001` 访问 API
- 通过在终端中按 <kbd>Ctrl + C</kbd> 随时停止服务器

服务器将继续运行，直到你停止它或关闭终端。
:::
