**页面**模块是一个只读活动模块，允许讲师使用富文本 HTML 创建静态教学内容。此模块类型非常适合呈现课程材料、讲义和其他信息内容。

## 概述

页面模块是信息性（只读）模块，呈现内容而不需要用户交互或提交。它们非常适合：

- 讲义和课程材料
- 阅读作业和补充内容
- 课程公告和重要信息
- 参考资料和文档

## 创建页面模块

要创建页面模块：

1. 导航到您的课程
2. 点击"添加模块"或"创建活动"
3. 从模块类型中选择"页面"
4. 输入页面的标题和描述
5. 使用富文本编辑器创建您的内容
6. 准备就绪后保存为草稿或发布

## 富文本编辑器

页面模块包含功能齐全的富文本编辑器，支持：

- **文本格式**：粗体、斜体、下划线、删除线
- **标题**：多个标题级别（H1-H6）
- **列表**：有序和无序列表
- **链接**：内部和外部链接
- **图像**：带有替代文本的内联图像
- **表格**：创建和格式化表格
- **代码块**：语法高亮的代码片段
- **HTML**：高级用户的直接 HTML 编辑

## 模块状态

页面模块遵循标准模块生命周期：

- **草稿**：内容正在创建和编辑；仅对创建者和授权用户可见
- **已发布**：内容已准备好供学生使用；对注册学生可见
- **已归档**：内容已保存但不再活跃；对学生隐藏

## 最佳实践

创建页面模块时：

- **使用清晰的标题**以层次结构组织内容
- **将长内容分段**以提高可读性
- **包含图像和图表**以增强理解
- **添加内部链接**以连接课程内的相关内容
- **使用列表**清晰地呈现信息
- **在发布前预览**以确保格式正确

## 编辑已发布的页面

页面模块发布后，您仍然可以编辑它。更改将在保存后立即对学生可见。考虑：

- 添加关于最近更新的说明以提高透明度
- 跟踪重要页面的更改
- 如果可用，使用版本历史来审查更改
