**白板**模块是一个只读活动模块，允许讲师创建以 Excalidraw JSON 格式存储的绘图和图表内容。此模块类型非常适合呈现视觉图表、流程图、思维导图和其他图形内容。

## 概述

白板模块是信息性（只读）模块，呈现视觉内容而不需要用户交互或提交。它们非常适合：

- 视觉图表和流程图
- 思维导图和概念图
- 技术图纸和示意图
- 交互式视觉解释

## 创建白板模块

要创建白板模块：

1. 导航到您的课程
2. 点击"添加模块"或"创建活动"
3. 从模块类型中选择"白板"
4. 输入白板的标题和描述
5. 使用 Excalidraw 编辑器创建您的视觉内容
6. 准备就绪后保存为草稿或发布

## Excalidraw 编辑器

白板模块使用 Excalidraw，这是一个功能强大的开源绘图工具，支持：

- **绘图工具**：笔、形状、箭头、线条
- **文本元素**：添加标签和注释
- **形状库**：矩形、圆形、菱形等
- **图层**：将元素组织到图层中
- **导出选项**：PNG、SVG 和 JSON 格式
- **协作**：实时协作编辑（适用于讲师）

## 模块状态

白板模块遵循标准模块生命周期：

- **草稿**：内容正在创建和编辑；仅对创建者和授权用户可见
- **已发布**：内容已准备好供学生使用；对注册学生可见
- **已归档**：内容已保存但不再活跃；对学生隐藏

## 最佳实践

创建白板模块时：

- **使用清晰的标签**以识别图表的不同部分
- **选择适当的颜色**以更好地进行视觉区分
- **保持图表简单**以避免让学生不知所措
- **在多个白板中使用一致的样式**
- **为复杂图表添加图例**
- **导出为图像**用于备份目的

## 查看白板

学生可以在交互式查看器中查看白板内容，允许：

- 放大和缩小以进行详细查看
- 平移以探索不同区域
- 将白板下载为图像
- 如果需要，打印白板
