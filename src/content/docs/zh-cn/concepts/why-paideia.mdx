**Paideia** 是为教育机构设计的现代、轻量的学习管理系统。Paideia 于 2025 年 9 月 29 日开始了成为下一代 LMS 的征程。使用最新的 Web 技术从头构建，Paideia 专注于简单性、性能和可维护性，同时提供全面的 LMS 功能。如果你需要一个易于部署、可轻松扩展并避免传统平台复杂性的 LMS，那么 Paideia 就是你的选择。

## 功能

**Paideia 是一个功能完整的学习管理系统。** 它包含管理课程、学生、作业、测验、讨论和成绩簿所需的一切。使用现代技术栈和稳健的设计决策构建，Paideia 在无技术债务的情况下提供企业级功能。

一些亮点包括：

- **[现代技术栈](/zh-cn/concepts/design-principles/#modern-tech-stack):** 使用 Bun、React Router v7 和 Elysia 构建，实现最佳性能。
- **[基于 React 的前端](/zh-cn/concepts/design-principles/#robust-design):** 基于 React 的坚实基础，使用 Mantine 组件提供一致、现代的 UI。
- **[功能完整](/zh-cn/concepts/design-principles/#feature-complete):** 完整的 LMS 功能，包括课程、注册、作业、测验、讨论和成绩簿。
- **[单二进制部署](/zh-cn/install-and-setup/):** 下载并运行单个可执行文件 — 无需复杂设置。
- **[可扩展的基础设施](/zh-cn/concepts/design-principles/#scalable-by-design):** 基于 PostgreSQL 和 S3 构建，占用空间小但可大规模扩展。
- **[无插件架构](/zh-cn/concepts/design-principles/#no-plugin-hell):** 与 Moodle 和 Canvas 不同，Paideia 避免了插件地狱和管理麻烦。
- **[LTI 支持](/zh-cn/concepts/design-principles/#lti-support):** 内置 LTI（学习工具互操作性）支持，用于与外部工具无缝集成。
- **[AI 原生](/zh-cn/concepts/design-principles/#ai-features):** 直接内置到平台的 AI 原生功能。
- **[Microsoft 集成](/zh-cn/concepts/design-principles/#microsoft-integration):** 内置 Microsoft 集成（Teams、OneDrive、Office 365）— 无需插件。

## 设计原则

Paideia 建立在六个核心设计原则之上，这些原则指导我们的开发并确保平台兑现其承诺。这些原则解释了我们为什么构建 Paideia、它解决的问题，以及为什么 Paideia 可能是你机构的最佳选择。

Paideia 是...

1. **现代技术栈，无技术债务：** 从一开始就使用最新技术和最佳实践构建。
2. **稳健的设计决策：** 基于 React 的坚实基础，具有生产级架构。
3. **功能完整：** 开箱即用的全面 LMS 功能。
4. **易于部署：** 单二进制部署 — 下载并运行。
5. **可扩展设计：** 基于 PostgreSQL 和 S3，占用空间小但可大规模扩展。
6. **无插件地狱：** 无插件架构 — 避免 Moodle 和 Canvas 的复杂性和管理麻烦。

## Paideia 背后的故事

**Paideia 由 [Hananoshika yomaru](https://yomaru.dev) 在加拿大 Trinity Western University 担任学习技术专家时创建。** 管理 Moodle 一直很痛苦 — 平台的复杂性使日常任务变得困难，通过插件扩展功能也有问题。大多数第三方 Moodle 插件缺乏适当的维护，存在错误，在安装前需要仔细考虑。这种插件依赖会产生技术债务和安全问题。

由于对这些限制感到沮丧，Paideia 从一个愿景中诞生：创建一个采用"开箱即用"方法并具有现代技术栈的 LMS。通过从一开始就选择正确的技术，Paideia 保持低技术债务，并为未来的漫长旅程做好准备。基本功能无需插件 — 所有必需的功能都是内置的、可维护的且安全的。

## 为什么选择 Paideia？

Paideia 代表了新一代学习管理系统。使用现代技术构建，设计易于部署，架构可扩展，Paideia 为你提供企业级 LMS 功能，而无需企业级复杂性。

无论你是小型机构还是大型大学，Paideia 的单二进制部署、可扩展的基础设施和功能完整的设计使其成为理想选择。你可以获得所需的所有功能，而无需传统平台的插件管理、技术债务或部署复杂性。
