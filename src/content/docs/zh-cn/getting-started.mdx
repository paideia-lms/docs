<CardGrid>
  <Card title="什么是 Paideia LMS？" icon="laptop">
    Paideia LMS 是一个现代、轻量的学习管理系统，使用 [Bun](https://bun.sh) 和 [React Router v7](https://reactrouter.com/) 构建。设计为单一可执行文件，极其易于管理和部署。
  </Card>

  <Card title="快速开始" icon="star">
    几秒内即可开始！下载预构建的二进制文件，并通过我们的[安装指南](/zh-cn/install-and-setup/)启动你的 LMS。或者[试用演示沙盒](https://demo.paideialms.com)无需安装即可探索 Paideia LMS。
  </Card>

  <SplitCard title="安装 Paideia" icon="rocket">
    <Tabs syncKey="paideia-install">
      <TabItem label="macOS ARM64">
        ```sh
        curl -L -o paideia https://github.com/paideia-lms/paideia/releases/latest/download/paideia-macos-arm64
        chmod +x paideia
        xattr -c paideia        
        ./paideia
        ```
      </TabItem>
      <TabItem label="Linux ARM64">
        ```sh
        curl -L -o paideia https://github.com/paideia-lms/paideia/releases/latest/download/paideia-linux-arm64
        chmod +x paideia        
        ./paideia
        ```
      </TabItem>
      <TabItem label="Linux x64">
        ```sh
        curl -L -o paideia https://github.com/paideia-lms/paideia/releases/latest/download/paideia-linux-x64
        chmod +x paideia
        ./paideia
        ```
      </TabItem>
    </Tabs>

    我们的[安装指南](/zh-cn/install-and-setup/)提供了在 macOS、Linux 上安装 Paideia 以及使用 Docker Compose 的逐步说明。
