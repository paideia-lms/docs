  <Card title="什麼是 Paideia LMS？" icon="laptop">
    Paideia LMS 是一個現代、輕量的學習管理系統，使用 [Bun](https://bun.sh) 和 [React Router v7](https://reactrouter.com/) 建構。設計為單一可執行檔，極易管理和部署。
  </Card>

  <Card title="快速開始" icon="star">
    幾秒內即可開始！下載預建的二進位檔，並透過我們的[安裝指南](/zh-tw/install-and-setup/)啟動你的 LMS。或者[試用演示沙盒](https://demo.paideialms.com)無需安裝即可探索 Paideia LMS。
  </Card>

  <SplitCard title="安裝 Paideia" icon="rocket">
    <Tabs syncKey="paideia-install">
      <TabItem label="macOS ARM64">
        ```sh
        curl -L -o paideia https://github.com/paideia-lms/paideia/releases/latest/download/paideia-macos-arm64
        chmod +x paideia
        xattr -c paideia        
        ./paideia
        ```
      </TabItem>
      <TabItem label="Linux ARM64">
        ```sh
        curl -L -o paideia https://github.com/paideia-lms/paideia/releases/latest/download/paideia-linux-arm64
        chmod +x paideia        
        ./paideia
        ```
      </TabItem>
      <TabItem label="Linux x64">
        ```sh
        curl -L -o paideia https://github.com/paideia-lms/paideia/releases/latest/download/paideia-linux-x64
        chmod +x paideia
        ./paideia
        ```
      </TabItem>
    </Tabs>

    我們的[安裝指南](/zh-tw/install-and-setup/)提供在 macOS、Linux 上安裝 Paideia 以及使用 Docker Compose 的逐步說明。
