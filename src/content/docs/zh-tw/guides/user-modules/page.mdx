**頁面**模組是一個只讀活動模組，允許講師使用富文本 HTML 創建靜態教學內容。此模組類型非常適合呈現課程材料、講義和其他信息內容。

## 概述

頁面模組是資訊性（只讀）模組，呈現內容而不需要用戶交互或提交。它們非常適合：

- 講義和課程材料
- 閱讀作業和補充內容
- 課程公告和重要信息
- 參考資料和文檔

## 創建頁面模組

要創建頁面模組：

1. 導航到您的課程
2. 點擊"添加模組"或"創建活動"
3. 從模組類型中選擇"頁面"
4. 輸入頁面的標題和描述
5. 使用富文本編輯器創建您的內容
6. 準備就緒後保存為草稿或發布

## 富文本編輯器

頁面模組包含功能齊全的富文本編輯器，支持：

- **文本格式**：粗體、斜體、下劃線、刪除線
- **標題**：多個標題級別（H1-H6）
- **列表**：有序和無序列表
- **鏈接**：內部外部鏈接
- **圖像**：帶有替代文本的內聯圖像
- **表格**：創建和格式化表格
- **代碼塊**：語法高亮的代碼片段
- **HTML**：高級用戶的直接 HTML 編輯

## 模組狀態

頁面模組遵循標準模組生命周期：

- **草稿**：內容正在創建和編輯；僅對創建者和授權用戶可見
- **已發布**：內容已準備好供學生使用；對註冊學生可見
- **已歸檔**：內容已保存但不再活躍；對學生隱藏

## 最佳實踐

創建頁面模組時：

- **使用清晰的標題**以層次結構組織內容
- **將長內容分段**以提高可讀性
- **包含圖像和圖表**以增強理解
- **添加內部鏈接**以連接課程內的相關內容
- **使用列表**清晰地呈現信息
- **在發布前預覽**以確保格式正確

## 編輯已發布的頁面

頁面模組發布後，您仍然可以編輯它。更改將在保存後立即對學生可見。考慮：

- 添加關於最近更新的說明以提高透明度
- 跟踪重要頁面的更改
- 如果可用，使用版本歷史來審查更改
