**白板**模組是一個只讀活動模組，允許講師創建以 Excalidraw JSON 格式存儲的繪圖和圖表內容。此模組類型非常適合呈現視覺圖表、流程圖、思維導圖和其他圖形內容。

## 概述

白板模組是資訊性（只讀）模組，呈現視覺內容而不需要用戶交互或提交。它們非常適合：

- 視覺圖表和流程圖
- 思維導圖和概念圖
- 技術圖紙和示意圖
- 互動式視覺解釋

## 創建白板模組

要創建白板模組：

1. 導航到您的課程
2. 點擊"添加模組"或"創建活動"
3. 從模組類型中選擇"白板"
4. 輸入白板的標題和描述
5. 使用 Excalidraw 編輯器創建您的視覺內容
6. 準備就緒後保存為草稿或發布

## Excalidraw 編輯器

白板模組使用 Excalidraw，這是一個功能強大的開源繪圖工具，支持：

- **繪圖工具**：筆、形狀、箭頭、線條
- **文本元素**：添加標籤和註釋
- **形狀庫**：矩形、圓形、菱形等
- **圖層**：將元素組織到圖層中
- **導出選項**：PNG、SVG 和 JSON 格式
- **協作**：實時協作編輯（適用於講師）

## 模組狀態

白板模組遵循標準模組生命周期：

- **草稿**：內容正在創建和編輯；僅對創建者和授權用戶可見
- **已發布**：內容已準備好供學生使用；對註冊學生可見
- **已歸檔**：內容已保存但不再活躍；對學生隱藏

## 最佳實踐

創建白板模組時：

- **使用清晰的標籤**以識別圖表的不同部分
- **選擇適當的顏色**以更好地進行視覺區分
- **保持圖表簡單**以避免讓學生不知所措
- **在多個白板中使用一致的樣式**
- **為複雜圖表添加圖例**
- **導出為圖像**用於備份目的

## 查看白板

學生可以在互動式查看器中查看白板內容，允許：

- 放大和縮小以進行詳細查看
- 平移以探索不同區域
- 將白板下載為圖像
- 如果需要，打印白板
