## 開始之前需要了解什麼？

如果你對**命令列介面**和**Web應用程式**有一些基本的了解，那麼你完全可以開始了！只需按照指示進行操作，你就可以完成整個教程。Paideia LMS 旨在讓所有人都能使用！🧑‍🚀 👩‍🚀 👨‍🚀

你需要：
- 運行 macOS 或 Linux 的電腦
- 基本的終端/命令列知識
- 存取設置 PostgreSQL 資料庫和 S3 相容儲存（或使用 Docker Compose）

<details>
<summary>如何使用每頁底部的清單？</summary>

把它們勾掉!

在每個頁面的末尾，你會發現一個可點擊的任務清單，其中列出了你現在應該能夠完成的任務。勾選這些項目，就可以在「教程追蹤」中看到你的進度。

使用清單不是必須的，但如果你在多次訪問中完成教程，它可以幫助你記住你的進度。你也可以留下一些任務不勾選，作為你稍後回顧的單元的提醒。

（這些資料僅保存在你的瀏覽器本地儲存中，在其他地方無法使用。它們不會被發送到其他地方，也不會由 Paideia LMS 儲存。）
</details>

<details>
<summary>我已經知道第一單元該做什麼了，我可以跳過嗎？</summary>

你可以使用[第一單元](/zh-tw/tutorial/1-setup/)裡面的課程，確保你擁有完成教程所需的工具和設置。它將指導你安裝 Paideia LMS、設置資料庫和儲存，以及配置你的環境。

如果你[已經安裝了 Paideia LMS](/zh-tw/install-and-setup/)並對你的設置感到滿意，你可以安全地跳到[第二單元](/zh-tw/tutorial/2-users/)，你將開始創建你的第一個課程。
</details>

<details>
<summary>如果我需要幫助，或者想了解更多關於 Paideia LMS 的內容，該怎麼辦？</summary>

進入討論區提問、分享想法或從社群獲得幫助。
</details>
