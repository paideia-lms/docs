現在您已創建了頁面模組，讓我們創建一個**白板**模組！白板非常適合創建視覺圖表、流程圖和其他圖形內容。

## 創建白板模組

<Steps>
1. 導航到您想要添加模組的模組庫或課程。

2. 點擊「添加模組」或「創建活動」按鈕。

3. 從模組類型列表中選擇「白板」。

   您應該看到以下欄位：
   - **標題** - 您的白板模組的名稱
   - **描述** - 白板內容的簡要說明
   - **白板編輯器** - 用於創建視覺內容的 Excalidraw 編輯器
</Steps>

## 創建視覺內容

<Steps>
4. 為您的白板模組輸入標題，例如：「系統架構圖」

5. 添加描述：「此白板顯示了 Paideia LMS 的架構。」

6. 使用 Excalidraw 編輯器創建您的視覺內容：

   Excalidraw 編輯器提供：
   - **繪圖工具**：筆、形狀、箭頭、線條
   - **文字元素**：添加標籤和註釋
   - **形狀庫**：矩形、圓形、菱形等
   - **圖層**：將元素組織到圖層中
   - **顏色**：為不同元素選擇顏色
   - **匯出選項**：PNG、SVG 和 JSON 格式

7. 創建一個簡單的圖表：
   - 繪製一個標記為「前端」的矩形
   - 繪製另一個標記為「後端」的矩形
   - 用箭頭連接它們
   - 添加文字標籤來描述連接
   - 使用顏色來區分不同的組件
</Steps>

## 保存並發布您的白板

<Steps>
8. 在預覽中查看您的白板內容。

9. 選擇您的模組狀態：
   - **草稿**：如果您想稍後繼續編輯，則保存為草稿
   - **已發布**：如果您準備好讓學生使用，則發布

10. 點擊「保存」或「發布」以保存您的白板模組。

11. 驗證您的白板是否出現在您的模組庫或課程中。
</Steps>

:::tip[白板最佳實踐]

- 使用清晰的標籤來識別圖表的不同部分
- 選擇適當的顏色以獲得更好的視覺區分
- 保持圖表簡單，以免讓學生感到不知所措
- 在多個白板之間使用一致的樣式
- 為複雜圖表添加圖例
:::

恭喜！您已創建了第一個白板模組！
