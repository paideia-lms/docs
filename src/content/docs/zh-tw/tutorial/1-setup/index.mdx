現在你已經知道你要搭建什麼了，是時候來設置你所需的所有工具！

本單元將向你展示如何安裝 Paideia LMS，設置資料庫和儲存，以及配置環境。如果你已經安裝並配置了 Paideia LMS，可以直接跳到[第二單元](/zh-tw/tutorial/2-users/)。

:::tip[安裝前先試用]

想在安裝前試用 Paideia LMS 嗎？訪問 [demo.paideialms.com](https://demo.paideialms.com) 無需安裝即可探索所有功能。

演示託管在 Hetzner CPX21 實例（3 vCPU、4GB RAM、80GB 本地磁碟）上，每日備份，每月成本不到 $10，展示了 Paideia LMS 的高效資源使用。
:::

## 本單元目標

在本單元中，你將**下載並安裝 Paideia LMS**，**設置資料庫和儲存**，以及**配置環境**。

你將學習如何：
- 安裝 Paideia LMS 二進位檔案
- 設置 PostgreSQL 資料庫
- 配置 S3 相容儲存
- 配置環境變數
- 存取你的 Paideia LMS 實例

安裝完成後，你就可以在下一單元中創建第一個課程了！
