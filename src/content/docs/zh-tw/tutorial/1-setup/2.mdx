## 下載 Paideia LMS 二進位檔案

單二進位安裝是啟動 Paideia LMS 最快的方法。Paideia LMS 作為包含所需一切的單個可執行檔案分發。

<Steps>
1. 下載適用於你平台的二進位檔案：

   <Tabs>
     <TabItem label="macOS ARM64">
       ```bash
       curl -L -o paideia https://github.com/paideia-lms/paideia/releases/latest/download/paideia-macos-arm64
       chmod +x paideia
       xattr -c paideia
       ```
     </TabItem>
     <TabItem label="Linux ARM64">
       ```bash
       curl -L -o paideia https://github.com/paideia-lms/paideia/releases/latest/download/paideia-linux-arm64
       chmod +x paideia
       ```
     </TabItem>
     <TabItem label="Linux x64">
       ```bash
       curl -L -o paideia https://github.com/paideia-lms/paideia/releases/latest/download/paideia-linux-x64
       chmod +x paideia
       ```
     </TabItem>
   </Tabs>

2. 透過檢查版本來驗證安裝：

   ```bash
   ./paideia version
   ```

   你應該看到 Paideia LMS 版本號，確認二進位檔案正常工作。
</Steps>

:::tip[替代方案：Docker Compose]

如果你更喜歡使用 Docker Compose 進行包含 PostgreSQL 和 MinIO 的完整設置，請參閱[安裝和設置指南](/zh-tw/install-and-setup/#docker-compose-安裝)中的 Docker Compose 說明。
:::
