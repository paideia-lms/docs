## 啟動 Paideia LMS

現在你的環境已經配置好了，你可以啟動 Paideia LMS 伺服器。

<Steps>
1. 啟動 Paideia LMS 伺服器：

   ```bash
   ./paideia server
   ```

   或者簡單地：

   ```bash
   ./paideia
   ```

   伺服器將啟動並在配置的埠上監聽（預設：後端 API 為 `3001`，前端為 `3000`）。

2. 等待伺服器啟動。你應該在終端中看到 Paideia LMS 正在運行的確認資訊。

   :::tip[伺服器啟動]
   如果有待處理的遷移，伺服器將在啟動時自動運行資料庫遷移。
   :::
</Steps>

## 存取 Paideia LMS

<Steps>
3. 打開瀏覽器並導航到 `http://localhost:3000`（或透過 `FRONTEND_PORT` 環境變數配置的埠）。

   你應該看到 Paideia LMS 登入頁面。

4. 首次存取時，系統會提示你創建管理員帳戶。按照螢幕上的說明創建管理員使用者。

   :::caution[首次設置]
   確保記住你的管理員憑證！你需要它們來登入和管理你的 Paideia LMS 實例。
   :::

5. 創建管理員帳戶後，登入以存取 Paideia LMS 儀表板。
</Steps>

恭喜！你現在已經運行並可以存取 Paideia LMS 了！

## 驗證安裝

<Steps>
6. 探索 Paideia LMS 介面：

   - 檢查儀表板
   - 瀏覽不同的部分
   - 驗證所有功能都可以存取

7. 檢查 API 端點：

   導航到 `http://localhost:3001` 以驗證 API 正在運行（你應該看到 API 文件或健康檢查回應）。
</Steps>

:::tip[運行 Paideia LMS]

當 Paideia LMS 伺服器正在運行時，你可以：
- 在 `http://localhost:3000` 存取 Web 介面
- 在 `http://localhost:3001` 存取 API
- 透過在終端中按 <kbd>Ctrl + C</kbd> 隨時停止伺服器

伺服器將繼續運行，直到你停止它或關閉終端。
:::
