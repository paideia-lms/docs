現在您有了帶章節的課程，是時候將您的活動模組添加到課程中了！模組連結到課程，允許您在多個課程中重用相同的模組。

<PreCheck>
  - 將活動模組連結到課程章節
  - 配置課程特定的模組設置
  - 在章節內重新排序模組
</PreCheck>

## 將模組連結到課程章節

<Steps>
1. 導航到要添加模組的章節。

2. 在章節中點擊「添加模組」或「連結模組」。

3. 從您的模組庫中選擇一個模組：
   - 您將看到您可以存取的模組列表
   - 選擇要添加到此章節的模組
   - 模組將連結到課程章節

4. 配置模組順序：
   - 指定模組在章節內的順序
   - 您稍後可以使用拖放或排序控件重新排序模組

   :::tip[模組可重用性]
   單個活動模組可以連結到多個課程。這意味著您可以創建一次模組並在不同課程中重用它，節省時間並確保一致性。
   :::
</Steps>

## 配置課程特定設置

<Steps>
5. 為模組配置課程特定設置：

   當模組添加到課程時，它成為具有課程特定設置的**課程模組**。這些設置允許您自訂模組在此特定課程中的行為方式。

6. 設置模組特定選項：
   - **自訂模組名稱**：為此課程覆蓋預設模組名稱
   - **截止日期**（用於互動式模組）：設置提交截止日期
   - **開放/關閉日期**：控制模組何時對學生可用
   - **截止日期**：允許延遲提交但帶有懲罰

   :::note[課程模組設置]
   課程模組設置僅影響模組在此特定課程中的顯示和行為方式。它們不會更改模組的預設配置或影響其在其他課程中的使用。
   :::

7. 保存您的課程模組設置。
</Steps>

## 添加多個模組

<Steps>
8. 向您的課程添加更多模組：

   - 重複該過程以向不同章節添加模組
   - 您可以將同一模組添加到多個章節或課程
   - 在每個章節內邏輯地組織模組

9. 根據需要重新排序模組：
   - 使用拖放或排序控件
   - 模組在每個章節內排序，控制學習順序
</Steps>

## 模組組織範例

您的課程現在可能如下所示：

```
Paideia LMS 簡介
├── 第1週：介紹
│   ├── 第1天：入門
│   │   ├── 歡迎頁面（頁面模組）
│   │   └── 課程概述測驗（測驗模組）
│   └── 第2天：基本概念
│       ├── 概念頁面（頁面模組）
│       └── 練習測驗（測驗模組）
└── 第2週：高級主題
    ├── 高級概念（頁面模組）
    └── 最終作業（作業模組）
```
