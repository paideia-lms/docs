現在您已經創建了活動模組並個性化您的資料，是時候將您的內容組織到課程中並管理學生存取了！

## 接下來

在本單元中，您將學習如何在 Paideia LMS 中創建和管理**課程**。您將構建：

- 具有組織好的章節和類別的課程
- 透過向章節添加模組來創建課程內容
- 透過註冊管理學生存取
- 使用小組組織學生

在此過程中，您將學習如何構建課程內容，管理學生存取，以及將學生組織到小組中進行協作活動。
