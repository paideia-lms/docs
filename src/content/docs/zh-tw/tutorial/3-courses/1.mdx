## 創建新課程

<Steps>
1. 導航到課程創建區域。通常可以透過在課程管理介面中點擊「創建課程」或「新建課程」找到。

2. 輸入課程資訊：
   - **標題**：為您的課程輸入描述性標題，例如：「Paideia LMS 簡介」
   - **描述**：添加課程涵蓋內容的簡要說明
   - **類別**（可選）：選擇或創建類別以組織您的課程

3. 點擊「創建」或「保存」以創建您的課程。

   :::tip[課程類別]
   類別有助於組織您的課程目錄。您可以創建嵌套類別以實現分層組織。例如，「電腦科學」 → 「程式設計」 → 「Web開發」。
   :::
</Steps>

## 使用章節組織您的課程

<Steps>
4. 創建課程章節以組織您的內容：

   章節提供了一種邏輯地構建課程內容的方法。您可以創建：
   - **頂級章節**：主要單元或週
   - **嵌套章節**：主章節內的子章節

5. 創建您的第一個章節：
   - 在您的課程中點擊「添加章節」或「新建章節」
   - 輸入章節名稱，例如：「第1週：介紹」
   - 可選地添加描述
   - 保存章節

6. 創建嵌套章節（可選）：
   - 在章節內，您可以創建子章節
   - 例如，在「第1週：介紹」內創建：
     - "第1天：入門"
     - "第2天：基本概念"

   :::tip[章節組織]
   章節可以嵌套以創建分層結構。這允許您在多個級別組織內容，例如週 → 天 → 主題。
   :::

7. 根據需要重新排序章節：
   - 使用拖放或排序控件來排列章節
   - 章節和模組在每個章節內排序，控制學生遇到內容的順序
</Steps>

## 課程結構

您的課程現在具有如下結構：

```
Paideia LMS 簡介
├── 第1週：介紹
│   ├── 第1天：入門
│   └── 第2天：基本概念
├── 第2週：高級主題
└── 第3週：最終專案
```

每個章節可以包含：
- **子章節**（嵌套的子章節）
- **活動模組**（頁面、測驗、作業等）
