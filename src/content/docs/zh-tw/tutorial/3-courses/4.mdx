現在您已經註冊了學生，讓我們將他們組織到小組中！小組支援協作活動、靈活的權限管理和更好的課程組織。

<PreCheck>
  - 在您的課程中創建小組
  - 在註冊期間將學生分配到小組
  - 分層組織小組
</PreCheck>

## 在您的課程中創建小組

<Steps>
1. 導航到課程小組管理介面。通常可以透過在您的課程中點擊「小組」或「管理小組」找到。

2. 點擊「創建小組」或「新建小組」。

3. 輸入小組資訊：
   - **名稱**：輸入描述性名稱，例如：「A組」或「專案團隊1」
   - **描述**（可選）：添加小組用途的簡要說明
   - **父小組**（可選）：如果要創建嵌套小組，請選擇父小組

4. 保存小組。

   :::tip[分層小組]
   小組可以透過父小組和子小組分層組織。這支援複雜的組織結構，例如：
   - "2024年秋季" → "A組" → "團隊1"
   - "實驗小組" → "週一實驗" → "A組"
   :::
</Steps>

## 將學生分配到小組

<Steps>
5. 在註冊期間將學生分配到小組：

   - 註冊學生時，您可以將他們分配到一個或多個小組
   - 學生可以同時屬於多個小組
   - 例如，學生可能同時屬於「A組」和「專案團隊1」

6. 將現有學生分配到小組：
   - 轉到註冊管理介面
   - 編輯學生的註冊
   - 添加或刪除小組成員資格
   - 保存更改

7. 查看小組成員資格：
   - 小組成員資格在註冊介面中顯示為彩色徽章
   - 這使得很容易看到每個學生屬於哪些小組
</Steps>

## 將小組用於課程活動

<Steps>
8. 了解小組如何支援課程活動：

   **小組作業**：
   - 為小組工作配置作業
   - 所有小組成員在單個提交中協作
   - 促進團隊合作並減少評分工作量

   **小組論壇**：
   - 將討論論壇設置為可見小組或分離小組模式
   - 學生可以看到其他小組的討論（可見小組）或僅看到自己的（分離小組）

   **小組測驗**：
   - 將測驗限制為小組成員
   - 支援基於小組的評估或協作測驗活動

   **模組存取控制**：
   - 根據小組成員資格控制哪些學生可以存取特定的課程模組
   - 向不同小組顯示不同內容
   - 創建小組特定的活動和資源
</Steps>

## 小組組織範例

您的課程小組可能如下組織：

```
2024年秋季課程
├── A組
│   ├── 團隊1
│   ├── 團隊2
│   └── 團隊3
├── B組
│   ├── 團隊1
│   └── 團隊2
└── 實驗小組
    ├── 週一實驗
    └── 週三實驗
```
