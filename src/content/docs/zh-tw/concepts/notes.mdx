筆記是一個個人日誌功能，允許用戶創建、編輯和管理自己的筆記。筆記以 Markdown 內容形式存儲，可以公開共享或保持私密。

## 什麼是筆記？

筆記是用戶可以創建的個人條目，用於記錄想法、觀點、觀察或其他內容。它們在 Paideia LMS 平台內像數字日誌或筆記本一樣工作。

每個筆記包含：
- **內容** - 筆記的主要內容（支持 Markdown 格式）
- **可見性** - 可以設置為公開或私密
- **日期** - 自動跟踪筆記創建和更新的時間

## 創建筆記

用戶可以隨時創建筆記。創建筆記時，您可以：

- **編寫內容**使用 Markdown 格式創建富文本
- **添加圖片**通過直接上傳到筆記中
- **設置可見性**使筆記公開或保持私密
- **隨時保存和編輯**您的筆記

筆記支持富文本格式，允許您創建包含標題、列表、鏈接和嵌入圖片結構化條目。

## 管理筆記

創建後，可以通過多種方式管理筆記：

### 查看筆記

您可以按日期組織的列表視圖中查看所有筆記。系統還提供：

- **活動熱圖** - 顯示您創建筆記時間的可視化日曆
- **日期過濾** - 按特定日期過濾筆記
- **日曆視圖** - 查看不同日期下的筆記活動

### 編輯筆記

您可以隨時編輯自己的筆記。編輯筆記時，您可以：

- **更新內容** - 修改文本和格式
- **更改可見性** - 在公開和私密之間切換
- **添加或刪除圖片** - 更新嵌入的圖片

### 刪除筆記

當不再需要時，您可以刪除自己的筆記。刪除的筆記無法恢復，因此在確認刪除之前，請確保您想要刪除它們。

## 公開和私密筆記

筆記可以設置為公開或私密：

### 私密筆記

- **默認設置** - 筆記默認為私密
- **只有您可以看到** - 私密筆記僅對您（和管理員）可見
- **個人日誌** - 使用私密筆記記錄個人想法和反思

### 公開筆記

- **與他人共享** - 公開筆記可以被任何登錄系統的用戶查看
- **可選共享** - 您可以選擇將特定筆記設為公開
- **社區參與** - 公開筆記允許您與社區分享見解和想法

## 個人資料中的筆記

筆記已集成到您的用戶資料中，允許您：

- **查看您的活動** - 查看您創建筆記的時間和頻率
- **跟踪您的進度** - 監控您隨時間推移的筆記習慣
- **與他人共享** - 在查看您的個人資料時讓他人看到您的公開筆記
