**Paideia** 是為教育機構設計的現代、輕量的學習管理系統。Paideia 於 2025 年 9 月 29 日開始了成為下一代 LMS 的征程。使用最新的 Web 技術從頭構建，Paideia 專注於簡單性、效能和可維護性，同時提供全面的 LMS 功能。如果你需要一個易於部署、可輕鬆擴展並避免傳統平台複雜性的 LMS，那麼 Paideia 就是你的選擇。

## 特色

**Paideia 是一個功能完整的學習管理系統。** 它包含管理課程、學生、作業、測驗、討論和成績簿所需的一切。使用現代技術棧和穩健的設計決策構建，Paideia 在無技術債務的情況下提供企業級功能。

一些亮點包括：

- **[現代技術棧](/zh-tw/concepts/design-principles/#modern-tech-stack):** 使用 Bun、React Router v7 和 Elysia 構建，實現最佳效能。
- **[基於 React 的前端](/zh-tw/concepts/design-principles/#robust-design):** 基於 React 的堅實基礎，使用 Mantine 元件提供一致、現代的 UI。
- **[功能完整](/zh-tw/concepts/design-principles/#feature-complete):** 完整的 LMS 功能，包括課程、註冊、作業、測驗、討論和成績簿。
- **[單一二進制部署](/zh-tw/install-and-setup/):** 下載並執行單個可執行檔 — 無需複雜設定。
- **[可擴展的基礎設施](/zh-tw/concepts/design-principles/#scalable-by-design):** 基於 PostgreSQL 和 S3 構建，佔用空間小但可大規模擴展。
- **[無插件架構](/zh-tw/concepts/design-principles/#no-plugin-hell):** 與 Moodle 和 Canvas 不同，Paideia 避免了插件地獄和管理麻煩。
- **[LTI 支援](/zh-tw/concepts/design-principles/#lti-support):** 內建 LTI（學習工具互通性）支援，用於與外部工具無縫整合。
- **[AI 原生](/zh-tw/concepts/design-principles/#ai-features):** 直接內建到平台的 AI 原生功能。
- **[Microsoft 整合](/zh-tw/concepts/design-principles/#microsoft-integration):** 內建 Microsoft 整合（Teams、OneDrive、Office 365）— 無需插件。

## 設計原則

Paideia 建立在六個核心設計原則之上，這些原則指導我們的開發並確保平台兌現其承諾。這些原則解釋了我們為什麼構建 Paideia、它解決的問題，以及為什麼 Paideia 可能是你機構的最佳選擇。

Paideia 是...

1. **現代技術棧，無技術債務：** 從一開始就使用最新技術和最佳實踐構建。
2. **穩健的設計決策：** 基於 React 的堅實基礎，具有生產級架構。
3. **功能完整：** 開箱即用的全面 LMS 功能。
4. **易於部署：** 單一二進制部署 — 下載並執行。
5. **可擴展設計：** 基於 PostgreSQL 和 S3，佔用空間小但可大規模擴展。
6. **無插件地獄：** 無插件架構 — 避免 Moodle 和 Canvas 的複雜性和管理麻煩。

## Paideia 背後的故事

**Paideia 由 [Hananoshika yomaru](https://yomaru.dev) 在加拿大 Trinity Western University 擔任學習技術專家時創建。** 管理 Moodle 一直很痛苦 — 平台的複雜性使日常任務變得困難，通過插件擴展功能也有問題。大多數第三方 Moodle 插件缺乏適當的維護，存在錯誤，在安裝前需要仔細考慮。這種插件依賴會產生技術債務和安全問題。

由於對這些限制感到沮喪，Paideia 從一個願景中誕生：創建一個採用"開箱即用"方法並具有現代技術棧的 LMS。通過從一開始就選擇正確的技術，Paideia 保持低技術債務，並為未來的漫長旅程做好準備。基本功能無需插件 — 所有必需的功能都是內建的、可維護的且安全的。

## 為什麼選擇 Paideia？

Paideia 代表了新一代學習管理系統。使用現代技術構建，設計易於部署，架構可擴展，Paideia 為你提供企業級 LMS 功能，而無需企業級複雜性。

無論你是小型機構還是大型大學，Paideia 的單一二進制部署、可擴展的基礎設施和功能完整的設計使其成為理想選擇。你可以獲得所需的所有功能，而無需傳統平台的插件管理、技術債務或部署複雜性。
