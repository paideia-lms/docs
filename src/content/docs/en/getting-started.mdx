<CardGrid>
  <Card title="What is Paideia LMS?" icon="laptop">
    Paideia LMS is a modern, lightweight Learning Management System built with [Bun](https://bun.sh) and [React Router v7](https://reactrouter.com/). Designed for extreme ease of management and deployment as a single executable.
  </Card>

  <Card title="Quick start" icon="star">
    Get up and running in seconds! Download the pre-built binary and start your LMS with our [installation guide](/en/install-and-setup/). Or [try the demo sandbox](https://demo.paideialms.com) to explore Paideia LMS without installation.
  </Card>

  <SplitCard title="Install Paideia" icon="rocket">
    <Tabs syncKey="paideia-install">
      <TabItem label="macOS ARM64">
        ```sh
        curl -L -o paideia https://github.com/paideia-lms/paideia/releases/latest/download/paideia-macos-arm64
        chmod +x paideia
        xattr -c paideia        
        ./paideia
        ```
      </TabItem>
      <TabItem label="Linux ARM64">
        ```sh
        curl -L -o paideia https://github.com/paideia-lms/paideia/releases/latest/download/paideia-linux-arm64
        chmod +x paideia        
        ./paideia
        ```
      </TabItem>
      <TabItem label="Linux x64">
        ```sh
        curl -L -o paideia https://github.com/paideia-lms/paideia/releases/latest/download/paideia-linux-x64
        chmod +x paideia
        ./paideia
        ```
      </TabItem>
    </Tabs>

    Our [installation guide](/en/install-and-setup/) has step-by-step instructions for installing Paideia on macOS, Linux, and using Docker Compose.
  </SplitCard>
