Now that you know what you're going to build, it's time to set up all the tools you'll need! 

This unit shows you how to install Paideia LMS, set up your database and storage, and configure your environment. Skip ahead to [Unit 2](/en/tutorial/2-users/) if you already have Paideia LMS installed and configured.

:::tip[Try before installing]

Want to try Paideia LMS before installing? Visit [demo.paideialms.com](https://demo.paideialms.com) to explore all features without installation.

The demo is hosted on a Hetzner CPX21 instance (3 vCPU, 4GB RAM, 80GB local disk) with daily backups, costing less than $10 per month, demonstrating Paideia LMS's efficient resource usage.
:::

## Where are you going?

In this unit, you will **download and install Paideia LMS**, **set up your database and storage**, and **configure your environment**. 

You'll learn how to:
- Install the Paideia LMS binary
- Set up PostgreSQL database
- Configure S3-compatible storage
- Configure environment variables
- Access your Paideia LMS instance

Once your installation is complete, you'll be ready to create your first course in the next unit!
