Now that you've created a page module, let's create a **Whiteboard** module! Whiteboards are perfect for creating visual diagrams, flowcharts, and other graphical content.

<PreCheck>
  - Create a whiteboard module

</PreCheck>

## Create a whiteboard module

<Steps>
1. Navigate to your module library or course where you want to add a module.

2. Click "Add Module" or "Create Activity" button.

3. Select "Whiteboard" from the list of module types.

   You should see a form with fields for:
   - **Title** - The name of your whiteboard module
   - **Description** - A brief description of the whiteboard content
   - **Whiteboard Editor** - The Excalidraw editor for creating visual content
</Steps>

## Create visual content

<Steps>
4. Enter a title for your whiteboard module, for example: "System Architecture Diagram"

5. Add a description: "This whiteboard shows the architecture of Paideia LMS."

6. Use the Excalidraw editor to create your visual content:

   The Excalidraw editor provides:
   - **Drawing tools**: Pen, shapes, arrows, lines
   - **Text elements**: Add labels and annotations
   - **Shapes library**: Rectangles, circles, diamonds, and more
   - **Layers**: Organize elements into layers
   - **Colors**: Choose colors for different elements
   - **Export options**: PNG, SVG, and JSON formats

7. Create a simple diagram:
   - Draw a rectangle labeled "Frontend"
   - Draw another rectangle labeled "Backend"
   - Connect them with an arrow
   - Add text labels to describe the connection
   - Use colors to distinguish different components
</Steps>

## Save and publish your whiteboard

<Steps>
8. Review your whiteboard content in the preview.

9. Choose your module status:
   - **Draft**: Save as draft if you want to continue editing later
   - **Published**: Publish if you're ready to make it available to students

10. Click "Save" or "Publish" to save your whiteboard module.

11. Verify your whiteboard appears in your module library or course.
</Steps>

:::tip[Whiteboard best practices]

- Use clear labels to identify different parts of diagrams
- Choose appropriate colors for better visual distinction
- Keep diagrams simple to avoid overwhelming students
- Use consistent styling across multiple whiteboards
- Add legends for complex diagrams
:::

Congratulations! You've created your first whiteboard module!
